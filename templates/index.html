Правила работы с селектором:

  • Селектор предназначен для подбора типовых опор крепления инженерных сетей и оборудования.
  • Доступ к функционалу открыт только зарегистрированным пользователям.
  • С альбомами чертежей типовых опор можно ознакомиться по ссылками ниже.
  • Для начала работы необходимо:
    • Авторизироваться под своей учетной записью (если не зарегистрированы - пройти регистрацию).
    • Выбрать тип инженерной сети, для которой необходимо подобрать опоры.
    • Пользуясь выпадающими меню заполнить поля, цифрами указать длину участку коммуникации в метрах. (Количество опор будет рассчитано автоматически в соответствии с СП5.13330.2021)
    • Повторить п.4.3. для всех участков трассы. Добавленные опоры в корректном количестве появятся в разделе «выбранные опоры».
    • В разделе «выбранные опоры» указать адрес и наименование объекта, на который производится подбор.
    • Скачать спецификацию в формате xls, оформленную в соответствии с ГОСТ.
  • В случае, если предложенные решения не удовлетворяют требованиям к узлам опор, задайте вопрос на специальной платформе Ask.Hilti, либо позвоните по номеру 8-800-700-52-52 для связи с закрепленным за Вашей организацией менеджером.
  • Предложения по улучшению сервиса ждем на почте: подборщикопор@google.com
